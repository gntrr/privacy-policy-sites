Privacy Policy - Golek Ongkir

Last updated: [2025-08-13]

Introduction

Golek Ongkir ("we", "our app") is committed to protecting user privacy. This Privacy Policy explains how we collect, use, and protect your information when using the Golek Ongkir mobile application.

Information We Collect

1. Information You Provide

  • Shipping Data: Origin and destination shipping information, package weight
  • Tracking Data: Package tracking numbers and selected courier
  • Phone Number: Last phone number (optional) for specific package tracking
  • Search Data: Location search queries (provinces, cities, districts)

2. Automatically Collected Information

  • Device Information: Device model, operating system, app version
  • Usage Data: Features used, app usage time
  • App Logs: Technical data for debugging and app improvement
  • Internet Connection Status: To ensure API connectivity

3. Third-Party Information

  • AdMob Data: Google AdMob may collect data for ad personalization
  • Shipping API Data: Information from third-party API services for shipping and tracking data

How We Use Information

Usage Purposes:

  1. Core Services:
    • Calculate shipping costs
    • Track package status
    • Search location information
    • Provide courier information
  2. Service Improvement:
    • Analyze app usage
    • Fix bugs and improve performance
    • Develop new features
  3. Advertising:
    • Display relevant ads through Google AdMob
    • Optimize advertising experience

Information Sharing with Third Parties

We share information with:

  1. API Service Providers:
    • Shipping data is sent to third-party APIs to obtain shipping and tracking information
    • This data is necessary for the app's core functionality
  2. Google AdMob:
    • Device and usage data for ad personalization
    • Follows Google AdMob privacy policies
  3. Technical Service Providers:
    • Log data for performance analysis and debugging
    • Only data necessary for app maintenance

We will NOT:

  • Sell your personal data
  • Share sensitive information without consent
  • Use data for purposes beyond those stated

Data Security

Security Measures:

  • Encryption: All API communications use HTTPS
  • Local Storage: Sensitive data is not permanently stored on device
  • Limited Access: Only authorized personnel can access data
  • Security Updates: App is regularly updated for security

Data Storage:

  • Search and usage data is temporarily stored for app performance
  • Data is not permanently stored on our servers
  • App cache can be cleared through device settings

User Rights

You have the right to:

  1. Access: Know what data we collect
  2. Correction: Request correction of inaccurate data
  3. Deletion: Request deletion of personal data
  4. Restriction: Limit processing of certain data
  5. Portability: Obtain a copy of your data

How to Exercise Rights:

  • Contact us via email: contact@wimaalif.my.id
  • Uninstall the app to remove local data
  • Use device privacy settings to limit access

Advertising and Tracking

Google AdMob:

  • The app uses Google AdMob to display advertisements
  • AdMob may collect data for ad personalization
  • You can adjust ad preferences through Google settings

Test Mode:

  • In development mode, we use test advertisements
  • No personal data is sent to ad networks in test mode

Ad Opt-out:

  • Use "Limit Ad Tracking" setting on iOS
  • Use "Opt out of Ads Personalization" on Android
  • Visit Google Ad Settings

Children's Usage

  • This app is not intended for children under 13 years old
  • We do not knowingly collect data from children under 13
  • If you know a child under 13 is using this app, please contact us
  • Parents can contact us to delete their child's data

Policy Changes

Change Notifications:

  • We will notify users of significant changes
  • Changes will be posted in the app or through updates
  • The "Last updated" date will be changed according to revisions

Continued Consent:

  • Using the app after changes means you agree to the new policy
  • If you disagree, you can stop using the app

Data Retention

Storage Periods:

  • Search Data: Automatically deleted after 30 days
  • App Logs: Stored for maximum 90 days for debugging
  • Local Cache: Can be cleared anytime by user
  • Analytics Data: Aggregated and anonymized after 12 months

Contact

If you have questions about this Privacy Policy, please contact us:

  • Email: contact@wimaalif.my.id

Jurisdiction

This Privacy Policy is governed by Indonesian law. Any disputes will be resolved through competent courts in Indonesia.

Note: This policy applies to the Golek Ongkir mobile application. Different privacy policies may apply for web services or other platforms.
